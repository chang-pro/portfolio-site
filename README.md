## Design Decisions

| Decision | Why |
|----------|-----|
| **Tan/brown/cream palette** | Stands out immediately — 99% of dev portfolios are dark blue or black |
| **CSS-only animations** | IntersectionObserver + CSS transitions. Zero layout shifts, no heavy animation libraries |
| **Dancing Script signature** | Personal branding touch — feels human, not template-generated |
| **Light + dark mode** | Respects system preference, smooth toggle via next-themes |
| **No animation libraries** | Framer Motion and GSAP are overkill for fade/slide transitions. Pure CSS is faster and lighter. |
