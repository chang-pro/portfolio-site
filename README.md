**A personal developer portfolio with a warm, elegant design, dark/light mode, and smooth scroll-reveal animations.** Built to showcase projects and skills with a focus on clean aesthetics and accessibility.

---

## Features

- **Warm Color Palette** — Tan, cream, and coffee tones with elegant contrast
- **Dark/Light Mode** — Seamless theme switching via next-themes
- **Scroll Animations** — CSS-only fade and slide transitions via IntersectionObserver
- **Handwritten Signature** — Dancing Script font for a personal touch
- **Responsive Design** — Mobile-first with clean breakpoints
- **Accessible** — Semantic HTML, keyboard navigation, proper ARIA labels

## Design Choices

| Element | Choice | Reason |
|---------|--------|--------|
| Color palette | Tan/brown/cream | Stands out from typical dark dev portfolios |
| Animations | CSS-only (no JS libraries) | Lightweight, performant, no layout shifts |
| Typography | Dancing Script + system fonts | Personal branding without sacrificing readability |
| Theme | Light + dark with smooth toggle | Respects user preference, accessible |

## Tech Stack

- **Framework:** React 19, TypeScript
- **Styling:** Tailwind CSS
- **Theming:** next-themes (system preference detection)
- **Animations:** IntersectionObserver + CSS transitions
- **Fonts:** next/font with Dancing Script

## Skills Demonstrated

- Frontend design with a distinctive visual identity
- Responsive, mobile-first development
- Accessibility-first approach (semantic HTML, keyboard nav)
- Performance optimization (CSS-only animations, no heavy libraries)
- Theme system implementation
